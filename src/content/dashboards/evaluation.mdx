**Differentiation:** Two clickable pathway cards (Exercise-induced / Infection-based) that expand to show trigger, onset, hallmark immune changes, hormones, and resolution timeline. Below that, an 8-row comparison table covering muscle soreness, fatigue, resting HR, fever, CRP, lymphocytes, NLR/SII/SIRI, and performance — colour-coded blue vs. red with row-hover highlighting. **Assessment:** A visual neck-check decision flowchart (YES branch → STOP + Level 1 labs; NO branch → reduce intensity → monitor → gradual return), followed by six clickable instrument cards — WURSS-11, session-RPE load index, TQR recovery scale, POMS mood profile, DALDA composite questionnaire, and morning HR/HRV — each expanding to show thresholds and application. **Diagnostics:** Three expandable diagnostic pyramid levels (1 = screening, 2 = extended, 3 = specialised immunology) with the specific tests listed per level including the trace element panel with the Alis hemoconcentration correction note. Below that, the 5-step return-to-sport protocol as numbered steps, and the full field evaluation checklist table with colour-coded domains, tools, and action thresholds.
