# Medicine in Sports and Exercise

This repository contains the content for the course "Medicine in Sports and Exercise". It includes lecture notes, tutorials, and interactive dashboards on topics relevant to the field.

# Developer Notes

* `npm install`: installs needed packages (like astro)

scripts are defined in  [package.json][./package.json] under `scripts` which can be run via `npm run scriptname` like:

* `npm run dev`: runs a local version of the website
* `npm run publish`: commits all changes to the repo + pushes them to github + deploys it to github-pages
